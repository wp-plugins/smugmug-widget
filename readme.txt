A simple widget for displaying the most recent public images from a SmugMug account.

== Description ==

This easy-to-use widget for WordPress will display the most recent public images from your SmugMug account. All images are displayed using the same width, and linked to its original SmugMug display/purchase page. Widget configuration is simple, and does not require any special authorization steps. 

== Installation ==

1. Download <i>SmugMug Widget</i> from the WordPress repository or from <a href="http://www.pmkelly.com/smugmug-widget">the official page</a> and unzip its contents.
1. Upload the <i>smugmug-widget</i> directory to your <i>wp-content/plugins/</i> directory.
1. Activate the plugin from your WordPress dashboard.
1. Locate the new <i>SmugMug Widget</i> in your WordPress widget page, and drag it into one of your site's widget areas.
1. Fill out the necessary information in the widget panel.
1. The RSS feed URL for your SmugMug account is available at the bottom of your SmugMug login page. You should see a link there named <i>Available Feeds</a>.  Click that link and select the RSS feed for <i>Recent Photos</i>.

== Frequently Asked Questions ==

= Where is the official page for this plugin/widget? =

You can find the official development page <a href="http://www.pmkelly.com/smugmug-widget">here</a>.

= Where do I find the SmugMug RSS feed to put in the widget? =

At the bottom of your SmugMug login page, you should see a link named <i>Available Feeds</a>.  Click that link and select the RSS feed for <i>Recent Photos</i>.

= Why do I not see any images? =

One common reason you may not see any photos is that this widget only works with publicly accessible photos on SmugMug.  If your albums are private, this widget will not be able to retrieve them.
